# 📈 Stock Market Prediction & Analytics Platform

##Overview

This project analyzes and predicts Apple (AAPL) stock prices using historical market data from Yahoo Finance. The project covers data collection, data analysis, feature engineering, machine learning, and visualization.

## Objectives

* Collect real-world stock market data
* Perform exploratory data analysis
* Calculate technical indicators
* Build a machine learning prediction model
* Visualize stock trends and predictions
* Create an interactive Power BI dashboard

## Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
* Yahoo Finance API (yfinance)
* Power BI
* Git & GitHub
